# Project Navigation Guide

This page is a quick map of the repository.

## Where to start

If you are new here:

1. Read [README.md](README.md).
2. Read [docs/START_HERE.md](docs/START_HERE.md).
3. Open [exercises/ex1_db_starter.py](exercises/ex1_db_starter.py).

## Folder map

### docs/

All guides, roadmaps, and progress notes.

### exercises/

The beginner practice files.

### components/

The system-building stage.

### reference/

Finished examples you can study and compare against.

### tools/

Scripts for launching and testing the project.

## Suggested learning order

1. Read the beginner guide.
2. Complete the exercises.
3. Read the system-building guide.
4. Build the authentication component.
5. Keep going only if you want more practice.

## Why the repo is organized this way

The structure is meant to reduce confusion:

- docs for reading
- exercises for practice
- components for building
- reference for comparison
- tools for running the project

That way, a beginner can always find the next step without guessing.
